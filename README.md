AUTOTALLER PRO

Sistema de gestión minimalista para talleres mecánicos. Administra citas, órdenes, clientes, repuestos y empleados en un solo lugar.



COMO EMPEZAR

Opción 1: En tu PC o Mac
1. Descarga el archivo index.html
2. Haz doble clic para abrirlo en el navegador
3. Listo

Opción 2: En iPhone o iPad (Lo más fácil)
1. Entra a GitHub (github.com)
2. Crea una cuenta si no tienes
3. Crea un repositorio nuevo llamado autotaller-pro
4. Sube el archivo index.html
5. Ve a Settings → Pages
6. Activa GitHub Pages
7. Tu app estará en vivo. Ejemplo: https://tunombre.github.io/autotaller-pro

PASOS PARA SUBIR A GITHUB (DESDE IPHONE)

1. Abre Safari → github.com
2. Crea cuenta (si no tienes)
3. Toca el + arriba a la derecha
4. "New repository"
5. Nombre: autotaller-pro
6. Marca "Public"
7. "Create repository"
8. "Add file" → "Create new file"
9. Nombre: index.html
10. Pega todo el código del archivo HTML
11. "Commit new file"
12. Ve a Configuración (Settings)
13. Busca "Pages" en la izquierda
14. Source: main branch
15. ¡Guardado!

Tu app estará en: https://tunombre.github.io/autotaller-pro

Funciona en cualquier dispositivo, en cualquier lugar.



COMO FUNCIONA SIN INTERNET

Todo se guarda en la memoria del navegador. Los datos desaparecen si limpias el historial. Si necesitas guardarlos permanentemente, necesitaremos conectarlo a una base de datos (siguiente paso).



PRÓXIMOS PASOS (AVANZADO)

- Conectar a una base de datos real (Firebase, Supabase)
- Sincronización automática con WhatsApp real
- App nativa para iOS
- Reportes en PDF
- Integración con calendarios
